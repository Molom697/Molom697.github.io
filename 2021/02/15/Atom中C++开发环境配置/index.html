Atom中 C++ 开发环境配置

Posted on 2021-02-15 Edited on 2021-02-16 In Atom

今天心血来潮，将 Atom 的配置“提上日程”，也算是圆一个高中时的梦，也算是对自己一个新的证明吧。

需要的材料

  • Atom 下载地址
  • MinGW下载地址 提取码：pe6x
     

配置 gcc/g++ 编译环境

  1. 打开刚下载的MinGW进行安装，建议不要更改默认安装地址。

  2. 安装完成后点击 “continue”,打开 MinGW Installation Manager。

  3. 在 mingw32-gcc-g++ 一行右键，选择 “Mark for Installation”。

  4. 点击页面左上角的 “Installation” —> “Apply Changes”。

  5. 点击 “Apply”，下载完成后点击 “Close” 关闭页面。

  6. 右键 “此电脑”，选择 “属性” -> “高级” -> “环境变量”。点击 “新建”，并输入 “C:\MinGW\bin”（如果自行更改了 MinGW 的安装路径请自行对照修改此处的路径）。然后，全部点“确定”关闭所有对话框。

  7. 重启电脑

安装 Atom 插件

有了本地的配置环境，现在需要一个桥梁把配置环境和 Atom 连接起来。插件的安装可在 “File” -> “Settings” -> “Install” 中通过搜索得到。

必装

  • 语法自动检测插件 linter-gcc

  • 语法自动检测插件 linter

  • 编译插件 gcc-make-run

  1. 要记得在 gcc-make-run 的 settings 中将 make Utility 改为 gmake
